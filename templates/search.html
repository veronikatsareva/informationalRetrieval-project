Let's find something to watch!

Instruction
There are 3 fields: for the query, for the index and for the integer number.

  • The query is any string that you aim to find in the movie corpus. The query is checked with an implemented spellchecker.
  • The index is type of indexing / vectorization that are supported in the search: BM-25, word2vec and fasttext. Keep in mind that these vectorizing methods give different results due to different algorithms.
  • The integer number is an amount of records that you want to receive (e.g. the default is top-10 records).
The results are sorted in descending order according to their similarity to the query (the first one is the most relevant and similar and etc.).
