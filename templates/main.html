Welcome to the MovieSearch!


What is it?


This is a website that implements searching system for a collection of documents about the movies and TV-shows. The corpus consists of 5277 records. The data was borrowed from Kaggle. Each record consists of the title, the release year, the IMDB score and a short annotation. The search instructions are provided on the search page.
